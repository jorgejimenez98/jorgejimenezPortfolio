## Init project

Clone repository:

```sh
$ git clone https://github.com/jorgejimenez98/jorgejimenezPortfolio
$ cd jorgejimenezPortfolio

```
Install dependencies using the next code:

```sh
$ $ npm install --save
```

Run project:

```sh
$ npm start
```
  
The project will be running on url `http://127.0.0.1:3000`
